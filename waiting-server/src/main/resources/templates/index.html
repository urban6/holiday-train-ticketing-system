예약 유의사항

  • 예약 화면은 하나의 인터넷 창에서만 접속할 수 있습니다. 여러 창에서 동시에 접속하면 대기 순번이 초기화될 수 있습니다.
  • 접속자가 많은 시간에는 대기열을 통해 순서대로 입장하게 됩니다. 새로고침을 반복해도 순번은 앞당겨지지 않습니다.
  • 결제까지 마쳐야 예약이 확정됩니다. 좌석 선점 상태로 일정 시간이 지나면 자동으로 해제됩니다.

대규모 트래픽과 동시성 제어를 연습하기 위한 학습용 프로젝트입니다. 실제 승차권을 판매하지 않으며, 어떤 실제 기관과도 관련이 없습니다.
